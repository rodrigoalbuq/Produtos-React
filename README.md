# Produtos React

Este repositório contém uma aplicação React (Vite) dentro da pasta `my-app`.

## Como rodar localmente

```bash
cd my-app
npm install
npm run dev
```

## Estrutura

- `my-app/` — código-fonte da aplicação
  - `src/` — componentes, páginas e estilos
  - `index.html`, `vite.config.js`, `package.json`

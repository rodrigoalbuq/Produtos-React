# Produtos React

Aplicação React (Vite) localizada em `my-app`

## Rodando localmente

```bash
cd my-app
npm install
npm run dev
```

## Build

```bash
cd my-app
npm run build
```

Os arquivos estáticos serão gerados em `my-app/dist`.


## Estrutura

- `my-app/` — código-fonte da aplicação
  - `src/` — componentes, páginas e estilos
  - `index.html`, `vite.config.js`, `package.json`
